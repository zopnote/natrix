A package for creating command line interface applications in Dart.

It comes with a built-in structure for managing and creating command trees and their flags. 
Arguments are automatically parsed and passed securely. Messages, as well as standard input and output, 
are straightforward to handle, making this package a solid foundation for CLI applications.

Install with:
````bash
$ dart pub add natrix
````
